# Sistema de Textos Centralizados

Este diretório contém todos os textos do aplicativo de forma centralizada e tipada.

## Como usar

### Importação básica

```tsx
import texts from "../i18n/texts";

// Uso em componentes
<h1>{texts.home.title}</h1>
<p>{texts.home.subtitle}</p>
```

### Benefícios

1. **IntelliSense completo**: O TypeScript fornece autocompletar para todos os textos
2. **Type-safe**: Erros de digitação são detectados em tempo de compilação
3. **Centralizado**: Todos os textos em um único lugar, fácil de manter
4. **Preparado para i18n**: Estrutura pronta para adicionar múltiplos idiomas no futuro

### Estrutura

O objeto `texts` está organizado por seções da aplicação:

- `app`: Informações gerais do app
- `common`: Textos comuns usados em vários lugares
- `home`: Textos da página inicial
- `login`: Textos da página de login
- `mobileGuard`: Textos do componente de guarda mobile

### Exemplos de uso

```tsx
// Texto simples
{
  texts.login.title;
}

// Texto aninhado
{
  texts.login.form.email.placeholder;
}

// Ícones
{
  texts.home.features.bookFields.icon;
}

// Múltiplas propriedades
<input
  placeholder={texts.login.form.email.placeholder}
  aria-label={texts.login.form.email.label}
/>;
```

### Adicionando novos textos

1. Abra o arquivo `texts.ts`
2. Adicione o novo texto na seção apropriada
3. O TypeScript automaticamente irá disponibilizar o IntelliSense

```typescript
export const texts = {
  // ... outras seções
  myNewSection: {
    title: 'Meu Título',
    description: 'Minha descrição',
  },
} as const;
```

### Preparado para internacionalização

Esta estrutura facilita a adição de múltiplos idiomas no futuro:

```typescript
// Futuro: texts-pt.ts, texts-en.ts, texts-es.ts
export const texts = {
  pt: {
    /* textos em português */
  },
  en: {
    /* textos em inglês */
  },
  es: {
    /* textos em espanhol */
  },
};
```
